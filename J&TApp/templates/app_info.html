Proposed Database Application:

The database application we plan to develop is for a Car Rental Company primarily using MySQL for the database side. For the GUI we plan on developing it using ReactJS. This database will store the account information (name, payment info, address, phone number, age, email, account type), information on the cars (make, model, year, condition, color, price), each rental a customer makes as well as the condition of the car when it is returned, customer feedback/review for each rental, as well as favorites so customers can save favorite cars, and there will be an admin that can view all of this information as well as analytics based off of it.

Functionality Specifications:

Since we are choosing to develop a Car Rental Company database, the particular functionalities that we intend to implement are as follows:

  • Account creation that contains the creation date, name, age, address, phone number, and email.
  • User login information that contains attributes like username, password, and other relevant information about the user.
  • An account system that will track the information of an admin, who is in charge of all the rental listings, and the information of customers.
  • An admin dashboard where they can edit all the car listings, finalize the reservations made by customers, and track all the transactions.
  • The admin dashboard will display important analytics about requests, reservations, and information about the customers.
  • A notification system will be implemented to notify the admin about the reservations made by the customers.
  • A customer dashboard where customers can view the currently available cars as well as the description, and general attributes of the cars. The customer can select any car that shows available and reserve it instantly as long as the age requirement is met, upon which they will be charged.
  • A window to confirm all the personal details of the customer and payment information.
  • A profile information where customers can access to edit their personal information, payment type, and preferences.
  • A review/rating system for the experience of the customer with the car they rented.
  • A system that notifies the admin if the condition of a car after being returned is lower than when it was rented, and can charge the customer extra if that is the case.
